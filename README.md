# Sistema-de-Analisis-Farmacoligico-Automatizado-con-LLM-y-PubMed-API---LLM-
Sistema que integra la API de AEMPS, modelos LLM y PubMed para analizar medicamentos de forma automatizada. Genera resúmenes técnicos, traduce principios activos a nomenclatura internacional y selecciona evidencia científica relevante, creando informes estructurados en PDF listos para uso clínico o exploratorio.
